# English

## Overview

This repository contains a series of hard-surface modeling studies created while
following the **BlenderBros “Beginner Guide to 3D Modeling in Blender”** course.

The focus of this project is not speed, but building a **strong foundation** in:
- form design
- clean hard-surface topology
- structured modeling workflows
- basic lighting and presentation

All models were created from scratch in Blender.

## Project Focus

- Hard-surface modeling fundamentals
- Blockout → refinement → cleanup workflow
- Clean and readable sci-fi forms
- Boolean-based modeling
- Basic materials and lighting

Each commit represents a **completed logical step** in the workflow.

# Русский

## Обзор

выполненные в рамках курса **BlenderBros “Beginner Guide to 3D Modeling in Blender”**.

Основной фокус проекта:
- формирование правильного мышления в моделировании
